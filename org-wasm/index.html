Org-Rust

A parser & document exporter for Org-Mode. Progress is documented at the Org-Rust repository.

Editor

Try out the live editor below! The syntax highlighting is generated from the output of the parser ;)


            

        

    


    
More Info

    
    
        
  • The editor is powered by Web-Assembly and executes the parser directly from your browser!
    • 
        
  • The parser is zero-copy doesn't duplicate the source string in the generated AST.
    • 
        
  • Parsing relies on caching any successful sub-parses to minimize backtracking.
    • 
        
  • Nodes are stored in an index-based arena (a Vec) for more straightforward
            iteration and lifetime management.
